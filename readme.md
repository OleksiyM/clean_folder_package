# Homework 7 - Creating package in the Python

## build

Windows:
```
pip install build

```
Mac/Linux:
```
pip3 install build

```

```
py -m build .
```

Mac/Linux:
```
pytho3 -m build .
```

```
pip3 install sort-0.1.0-py3-none-any.whl 

```
