# Homework 7 - Creating package in the Python

## build

Windows:
```
pip install build

```
Mac/Linux:
```
pip3 install build

```
Windows:
```
py -m build .
```

Mac/Linux:
```
python3 -m build .
```

## Install whl Package from dist folder:

Windows:
```
pip install sort-0.1.3-py3-none-any.whl 
```
Mac/Linux:
```
pip3 install sort-0.1.3-py3-none-any.whl 

```
